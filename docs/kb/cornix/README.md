### Standard Signal
```

LONG
Entry: 67000-67200
TP1: 68000
TP2: 69000
SL: 66000
Leverage: 10x
```

```
BTCUSDT LONG

Entry 1: 67000 (25%)
Entry 2: 66500 (25%)
Entry 3: 66000 (50%)

TP2: 69000 (25%)
TP3: 70000 (50%)

SL: 65000
Leverage: 10x
```
